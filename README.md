# Wooden Carvings Image Classification 🪵

A deep learning model that classifies wooden carving images into categories 
using Convolutional Neural Networks (CNN).

## 📌 Project Overview
This project was developed during my internship at DreamAI. It classifies 
wooden carving images into three categories:
- 🌸 Flower carvings
- 🍎 Fruit carvings  
- 🌿 Leaf carvings

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- NumPy, Matplotlib
- Jupyter Notebook

## 📁 Project Structure
- `Image_Classification_Internship_Project.ipynb` — Full model training pipeline
- `trained_model.h5` — Trained model weights *(not included due to file size)*
- `DATASET FINAL/` — Training data (FLOWER, FRUIT, LEAF categories)
- `TESTING DATA/` — Test images for evaluation

## 📊 Results
- Successfully trained CNN model for wooden carving recognition
- Dataset includes custom labeled images across 3 categories
- Achieved accurate object recognition for cultural artifact identification

## 🚀 How to Run
1. Clone this repository
2. Open the `.ipynb` file in Jupyter Notebook
3. Install dependencies: `pip install tensorflow numpy matplotlib`
4. Run all cells in order
